# Group Project - BUS2

A Flask Web Application for sustainable food choices

## Setup

1. Create a new Project on PyCharm
2. Clone the repository - click on the terminal and copy the following commands
   1. git clone https://github.com/UOB-BUS-2/GroupProject
   2. cd GroupProject
3. Create your virtual environment with this terminal command
    1. python -m venv .venv
4. Activate the venv by:
   1. (For windows) .venv\Scripts\activate
   2. (For Mac/Linux) source .venv/bin/activate
5. Install the dependencies (packages you need for the app) 
   1. pip install -r requirements.txt
6. Run the application
   1. flask --app app run

## Saving Your Contributions
Here are the commands you need to put in the terminal one at a time to save your work to the Github Repo

1. git add .
2. git commit -m "Write a comment of what you did"
3. git push

## Opening the Latest Project

Lets say someone makes a change and you want to look at the latest version of the repository (code folder) so you can add to it

1. cd GroupProject
2. git pull

Note: Always do this every time (only do the setup once) before you code anything just so you work on the latest folder
